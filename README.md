# 🐍 Python Lab Programs — 1st Year

A complete collection of Python lab experiments and programs completed during **1st Year (Semester 1)**, covering all core topics taught in the curriculum — from basic syntax to object-oriented programming.

> Also includes the **semester project**: a menu-driven [Traffic Data Analysis System](#-semester-project).

---

## 📚 Topics Covered

| # | Topic | Concepts Practiced |
|---|---|---|
| 1 | **Basic I/O & Operators** | `input()`, `print()`, arithmetic, comparison, logical operators |
| 2 | **Control Flow** | `if`, `elif`, `else`, nested conditions |
| 3 | **Loops** | `for`, `while`, `break`, `continue`, `range()` |
| 4 | **Strings** | Slicing, methods (`.upper()`, `.split()`, `.replace()`), formatting |
| 5 | **Lists** | Indexing, slicing, `.append()`, `.sort()`, list comprehensions |
| 6 | **Tuples & Sets** | Immutability, set operations (union, intersection, difference) |
| 7 | **Dictionaries** | Key-value pairs, `.items()`, `.get()`, nested dictionaries |
| 8 | **Functions** | `def`, parameters, return values, default args, `*args`/`**kwargs` |
| 9 | **File Handling** | Read, write, append modes — `.txt` and `.csv` files |
| 10 | **Exception Handling** | `try`, `except`, `finally`, custom error messages |
| 11 | **OOP (Classes & Objects)** | Class, `__init__`, instance methods, object creation |
| 12 | **Modules & Libraries** | `os`, `csv`, `math`, `random`, `numpy` |

## ⚙️ Setup & Requirements

**Python version:** 3.x  
**External library:** `numpy` (only required for the project)

```bash
# Install numpy (only needed for the Traffic Analysis project)
pip install numpy

# Run any lab program
python 01_basic_io/hello_world.py
```

No other setup needed — all other programs use Python's built-in standard library.
